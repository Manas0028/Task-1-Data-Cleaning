# Customer Personality Analysis - Data Cleaning & Preprocessing

## Overview

This project demonstrates the data cleaning and preprocessing process using the **Customer Personality Analysis** dataset. The objective was to transform a raw dataset into a clean, structured, and analysis-ready format by identifying and resolving common data quality issues.

This project was completed as part of a **Data Analyst Internship – Task 1: Data Cleaning and Preprocessing**.

---

## Objective

The primary goal of this project is to:

* Identify and handle missing values
* Remove duplicate records
* Standardize categorical data
* Convert date columns into a consistent format
* Rename column headers for better readability
* Verify and correct data types
* Prepare the dataset for further analysis and visualization

---

## Dataset

**Dataset Name:** Customer Personality Analysis

The dataset contains customer demographic information, purchasing behavior, campaign responses, and spending patterns across different product categories.

## Data Cleaning Steps

### 1. Missing Values

* Identified missing values in the **Income** column.
* Removed rows containing missing income values.

### 2. Duplicate Records

* Checked the dataset for duplicate rows.
* Removed duplicate records to ensure data integrity.

### 3. Standardized Text Values

* Trimmed unnecessary spaces.
* Standardized text formatting for categorical columns such as:

  * Education
  * Marital_Status

### 4. Date Formatting

* Converted the **Dt_Customer** column into a consistent date format (DD-MM-YYYY).

### 5. Column Renaming

* Renamed column headers using lowercase letters and underscores for improved readability.

Example:

| Original       | Updated        |
| -------------- | -------------- |
| Year_Birth     | year_birth     |
| Marital_Status | marital_status |
| Dt_Customer    | dt_customer    |

### 6. Data Type Validation

Verified that:

* Numeric columns contain numeric values.
* Date columns are formatted correctly.
* Text columns are standardized.
